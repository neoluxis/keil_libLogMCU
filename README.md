# Keil LibLogMCU (适用于 MCU 的日志库，使用 Keil 编译)

## 简介

本库是纯头文件，提供了数个日志输出函数

定义了接受参数的宏，调用用户在 MCU 平台定义好的 `printf()` 函数，通过串口或其他接口打印日志
